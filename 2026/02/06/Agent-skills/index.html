Agent skills

一、什么是Agent skills

Agent skils通过SKILL.md把提示词规范化，如同一个规范的说明书供AI调用

skills规范目录结构：skill可自定义名称(支持中文)但必须放到对应IDE的skill目录下如.claude/skills/my-skill/

二、skills实例

与AI初始对话时，AI只会对比SKILL.md里元数据的内容，符合才进一步阅读整个md

1.仅仅最核心结构

创建.claude/skills/帮我写作/SKILL.md

由此，一旦与ai对话时符合帮我写作的相关内容，AI就会根据此描述来一步步操作允许

2.进一步包含文档、代码、附件等资源

进一步要求AI查看参考文档目录中的内容

3.与MCP配合

AI靠skills仅能产生文本图片等，加入MCP就可进行更多的操作，关于MCP请参考点击跳转
